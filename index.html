Research Projects

Biped Trajectory Optimization

at IvLabs, VNIT Nagpur

With a team of 5 members and 3 mentors, implemented the Direct Shooting Method of Trajectory Optimization on Planar Five-Link Underactuated System. The Trajectories were generated by a Non-Linear Program in Python using the CasADi toolbox and as well as for some cases in MATLAB, using fmincon function. The cost function used was the sum of torque squared over the entire trajectory where non-linear constraints varied for different cases.

  • Generated Periodic Path for Human Gait on a Flat terrain in Python as well as in MATLAB
  • Generated Periodic Path for Ostrich Gait on a Flat terrain in Python.
  • Generated a Single Human Periodic Gait in Python as well as in MATLAB.
  • As a side task also generated a trajectory for CartPole swing up problem
